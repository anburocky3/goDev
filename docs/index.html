Configuration

Simple YAML Config,
Infinite Possibilities

Define your services, ports, and paths in a single config file. Swap versions, add custom binaries, or tweak settings — all without touching code.

Read Documentation 
# config.yaml — goDev
services:
  nginx:
    binary: "./tools/nginx/nginx.exe"
    port: 8080
  mysql:
    binary: "./tools/mysql/bin/mysqld.exe"
    port: 3306
  php:
    version: "8.4.19"
    cgi_port: 9000
domains:
  suffix: ".test"
  ssl: true
