# Security Policy

## Supported Versions

The following versions of goDev are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability

We take the security of goDev seriously. If you discover a security vulnerability, please follow these steps:

1. **Do not** open a public GitHub issue
2. Email your findings to [15264938+anburocky3@users.noreply.github.com](mailto:15264938+anburocky3@users.noreply.github.com)
3. Include a detailed description of the vulnerability and steps to reproduce it
4. Allow reasonable time for the issue to be addressed before any public disclosure

## What to Include in Your Report

- Vulnerability description
- Steps to reproduce
- Potential impact
- Suggested fix (if any)
- Your contact information for follow-up

## Response Timeline

- We aim to acknowledge receipt of vulnerability reports within **48 hours**
- We will provide regular updates on the progress of a fix
- You will be notified when the vulnerability has been resolved
- Public disclosure will be coordinated with you

## Preferred Languages

We welcome all reasonable languages. English is preferred for ease of communication, but we will do our best to understand reports in other languages.
